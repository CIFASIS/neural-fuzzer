Neural-Fuzzer

Learn & discover bugs in your software

Download .zip Download .tar.gz View on GitHub

Neural-Fuzzer is an experimental fuzzer designed to use Machine Learning to learn from a set of initial files. It uses long-short term memory (LSTM) to learn how sequences of bytes are structured. After it is trained, it will automatically generate corrupted or unexpected files and it will try to crash a given program. It follows the same technique as rnn-char and other experiments.
